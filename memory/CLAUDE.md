# JARVIS Memory System Instructions

At the start of every session:
- Read all files in the /memory directory: decision.md, people.md, preferences.md, and user.md.
- Load their contents into your working memory for context and personalization.

At the end of every session:
- Update these files with any new decisions, people, preferences, or user information learned during the session.
- Ensure all important context is persisted for future sessions.

Keep memory files organized by category and concise. Never lose important user context.
