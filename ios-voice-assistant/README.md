# iOS Voice Assistant App

A modern iOS app (Swift/SwiftUI) for real-time voice chat, speech-to-text, and text-to-speech, powered by a backend API. Features:
- Voice chat interface
- Speech-to-text (backend-powered)
- Text-to-speech
- Real-time conversation UI
- Microphone and audio playback
- Modern, clean UI
- Push notifications (if possible)
- Accessibility and performance best practices

## Getting Started
1. Open the ios-voice-assistant folder in Xcode.
2. Add all Swift files, Info.plist, and .gitignore to the project.
3. Set your bundle identifier and signing settings.
4. Configure your backend API endpoint in SpeechAPI.swift.
5. Build and run on iOS device or simulator.

## Features
- Voice chat with backend-powered speech recognition
- TTS playback
- Conversation history
- Push notifications (optional)

---

Usage:
- Tap the microphone to record your voice.
- The app sends audio to the backend for transcription and TTS.
- Conversation history is shown in the UI.
