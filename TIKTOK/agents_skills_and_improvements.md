# Agents, Required Skills, and Improvement Suggestions

## 1. VideoIdeasPlanner Agent
**Purpose:** Generate a 30-day TikTok video ideas plan for a specified niche.

**Required Skills:**
- Content ideation and creativity
- Niche research and trend analysis
- Basic user interaction (prompting for input)
- List formatting and output

**Improvement Suggestions:**
- Integrate with TikTok trend APIs or web scraping for real-time trends
- Add AI-powered idea generation (e.g., using GPT)
- Suggest hashtags, captions, or posting times
- Allow user customization and feedback for each idea
- Export plans to CSV, PDF, or calendar formats

---

## 2. VideoCreator Agent
**Purpose:** Assist in creating TikTok videos based on the video ideas plan.

**Required Skills:**
- Step-by-step workflow guidance
- Script and storyboard generation
- Asset management (music, images, clips)
- Video editing basics (integration or instructions)
- User progress tracking

**Improvement Suggestions:**
- Integrate with video editing tools/APIs (e.g., CapCut, FFmpeg)
- Provide AI-assisted script writing and editing
- Offer templates for different video styles
- Automate asset sourcing (royalty-free music, stock footage)
- Add export/upload automation to TikTok
- Track and visualize progress for each video

---

You can expand these agents by adding more automation, AI features, integrations, and user customization options as your workflow evolves.
