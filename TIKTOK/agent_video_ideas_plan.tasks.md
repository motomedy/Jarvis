# VideoIdeasPlanner Agent Tasks

## Purpose
Generate a 30-day TikTok video ideas plan for a specified niche.

## Tasks
1. Prompt the user to enter their TikTok niche.
2. Generate 30 unique video ideas tailored to the niche.
3. Ensure ideas are varied (e.g., tips, challenges, trends, tutorials, Q&A, behind-the-scenes).
4. Output the plan in a clear, day-by-day format.
5. Optionally, allow exporting the plan to a file (CSV or TXT).
6. Optionally, suggest hashtags or captions for each idea.
7. Optionally, allow the user to regenerate or customize specific days.

## Example Output
```
Day 1: [Niche] Introduction & Story
Day 2: [Niche] Quick Tips
Day 3: [Niche] Challenge
...
Day 30: [Niche] Recap & Next Steps
```

Feel free to expand or modify these tasks as your needs evolve.
