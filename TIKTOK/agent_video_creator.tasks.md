# VideoCreator Agent Tasks

## Purpose
Assist in creating TikTok videos based on the 30-day video ideas plan.

## Tasks
1. Accept a video idea (or script) as input.
2. Guide the user through the video creation workflow:
   - Scripting
   - Asset selection (music, images, clips)
   - Recording tips
   - Editing steps
   - Adding effects, captions, and hashtags
3. Output a checklist or step-by-step guide for each video.
4. Optionally, integrate with video editing tools or APIs.
5. Optionally, allow exporting the video plan or script.
6. Optionally, track progress for each video idea.

## Example Output
```
Video Idea: [Idea]
- [ ] Write script
- [ ] Gather assets
- [ ] Record video
- [ ] Edit and add effects
- [ ] Add captions/hashtags
- [ ] Export and upload
```

Expand or modify these tasks as your workflow evolves.
