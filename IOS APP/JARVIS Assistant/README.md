# JARVIS Assistant

A native iOS app built with Swift and SwiftUI to replicate the features of the JARVIS web app.

## Features (Scaffolded)
- Voice Chat (push-to-talk, backend AI integration)
- Calendar integration
- Notes
- Mail (read-only)
- Persistent memory

## Structure
- Modern, minimal UI
- Navigation between features
- Placeholder views for each feature
- Voice chat interface: record audio, send to backend, display responses
